Trade Consolidator
Webapp para consolidar e analisar trades do NinjaTrader com Supabase backend.

Funcionalidades
✅ Login/Register com Supabase
✅ Upload de CSV do NinjaTrader
✅ Detecção automática de duplicatas
✅ Cálculo de PnL em pontos e dólares
✅ Gráficos interativos
✅ Dashboard com insights

Como usar
Acesse: https://trade-consolidator.vercel.app

Registre uma conta

Faça login

Upload seu CSV

Analise seus trades!

Tecnologia
Supabase (Backend + Auth)

JavaScript Vanilla

Chart.js (Gráficos)

Vercel (Hosting)

Instrumentos Suportados
NQ: 20 dólares/ponto

MNQ: 2 dólares/ponto

GC: 100 dólares/ponto

MGC: 10 dólares/ponto

Deploy
Hospedado no Vercel. Deploy automático a cada commit no GitHub.
